# Crop-Damage-Assessment-Application
A Mobile application called “FarmBuddy” ,a Crop-Damage-Assessment-Application to submit the compensation claims together with evidence of damage (E.g. photos/videos), which allows the crop damage compensation process to be more effective and less fraudulent.

A list of technologies used within the project:
* Flutter (https://flutter.dev/)
* Firebase(https://firebase.google.com/)
